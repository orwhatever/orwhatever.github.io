Simpe WebGL painting app

Sorry; your web browser does not support HTML5’s canvas element.

In order to reduce the calls to drawArrays the vertices are assigned incremental vertex numbers and are assigned to three different arrays for point, triangles, circles. Using these arrays. the render function is able to call once for each drawing mode session converning points and triangles. When someone is Point-Drawing mode all point are drawn with one call. When he/she is an Triangle-Drawing mode all triangles are rendered with one call. However this is not possible for circles for which separate triangle fans have to be drawn.
